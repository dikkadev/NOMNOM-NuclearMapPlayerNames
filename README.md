# NuclearMapPlayerNames

This repository is a GitHub release mirror for **NuclearMapPlayerNames**.

The source code and development history live in the Forgejo monorepo:

- https://forge.dikka.dev/lab/nuclear_option_mods

This mirror exists so NOMNOM/NOMM can use GitHub Releases for automatic mod updates.
It is not the canonical source repository.

## Mod

- NOMNOM id: `dev.dikka.nuclearmapplayernames`
- Tags: QoL, Map, Multiplayer
- Author: dikkadev

Client-side Nuclear Option mod that shows multiplayer usernames next to player-controlled aircraft on the dynamic map.

## Installation

Use NOMM/NOMNOM when possible. Manual releases contain one archive with the BepInEx plugin folder inside.
